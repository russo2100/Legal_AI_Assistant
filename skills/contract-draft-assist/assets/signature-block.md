## Рекомендации по оформлению

### Расположение
- Подписи сторон располагаются на одной странице
- Если не помещается — на следующей странице с заголовком «АДРЕСА И ПОДПИСИ СТОРОН»
- Для многостраничных документов — прошивка и нумерация

### Печати
- Для ООО: печать ставится при наличии (не обязательно с 2015 года)
- Для ИП: печать не требуется
- Печать не должна закрывать подпись

### Количество экземпляров
- Минимум 2 (по одному для каждой стороны)
- Для суда: количество по числу лиц + 1 для суда
- Для госорганов: по требованию
