# Требования к оформлению документов

## Общие требования

### Шрифты и форматирование
- **Шрифт:** Times New Roman, 14 pt (для договоров), 12-14 pt (для исков)
- **Межстрочный интервал:** 1,5
- **Поля:**
  - Левое: 25-30 мм
  - Правое: 10-15 мм
  - Верхнее: 20 мм
  - Нижнее: 20 мм
- **Абзацный отступ:** 1,25 см
- **Нумерация страниц:** снизу по центру или справа

## Структура договора

### Титульная часть (не нумеруется)
```
                    ДОГОВОР ПОДРЯДА № ___
                    
                         г. Москва              «___» ______ 20__ г.
```

### Преамбула
```
[Полное наименование организации], именуемое в дальнейшем «Заказчик», 
в лице [должность, ФИО], действующего на основании [Устава/доверенности], 
с одной стороны, и

[Полное наименование], именуемое в дальнейшем «Исполнитель», в лице 
[должность, ФИО], действующего на основании [Устава/доверенности], 
с другой стороны,

заключили настоящий Договор о нижеследующем:
```

### Разделы и нумерация
```
1. ПРЕДМЕТ ДОГОВОРА
   1.1. Исполнитель обязуется...
   1.2. Заказчик обязуется...

2. СТОИМОСТЬ И ПОРЯДОК РАСЧЁТОВ
   2.1. Стоимость работ составляет...
   2.2. Оплата производится...
```

### Подписи сторон
```

[Наименование]                     [Наименование]
Адрес: ___________                 Адрес: ___________
ИНН/КПП: ___________               ИНН/КПП: ___________
Р/с: ___________                   Р/с: ___________
Банк: ___________                  Банк: ___________
БИК: ___________                   БИК: ___________

____________/____________          ____________/____________
  Подпись   ФИО                       Подпись   ФИО

М.П.                               М.П.
```

## Структура искового заявления

### Шапка (правый верхний угол)
```
В Арбитражный суд города Москвы
107078, г. Москва, ул. Богдан Хмельницкий, д. 11

Истец: ООО «Ромашка»
Адрес: 123456, г. Москва, ул. Ленина, д. 1
ИНН: 7701234567, ОГРН: 1157700123456
Тел.: +7 (495) 123-45-67

Ответчик: ООО «Вектор»
Адрес: 109012, г. Москва, ул. Тверская, д. 2
ИНН: 7702345678, ОГРН: 1167700234567

Цена иска: 500 000 (пятьсот тысяч) рублей
Госпошлина: 7 000 (семь тысяч) рублей
```

### Заголовок
```
ИСКОВОЕ ЗАЯВЛЕНИЕ
о взыскании задолженности по договору
```

### Описательная часть
```
«___» ______ 20__ г. между Истцом и Ответчиком был заключён 
договор № ___ (далее — Договор).

По условиям Договора, Истец обязался выполнить работы, а Ответчик 
— оплатить их в размере ___ рублей в срок до «___» ______ 20__ г.

Истец свои обязательства исполнил полностью, что подтверждается 
Актом сдачи-приёмки от «___» ______ 20__ г.

Ответчик оплату не произвёл, несмотря на неоднократные требования.
```

### Мотивировочная часть
```
Согласно ст. 309 ГК РФ, обязательства должны исполняться надлежащим 
образом в соответствии с условиями обязательства и требованиями закона.

В силу ст. 310 ГК РФ, односторонний отказ от исполнения обязательства 
не допускается.

На основании ст. 330 ГК РФ, кредитор вправе требовать уплаты 
неустойки за просрочку исполнения обязательства.
```

### Просительная часть
```
На основании изложенного, руководствуясь ст. 309-310, 330 ГК РФ, 
ст. 125-126 АПК РФ,

ПРОШУ:

1. Взыскать с ООО «Вектор» в пользу ООО «Ромашка» задолженность 
   по договору в размере 500 000 (пятьсот тысяч) рублей.

2. Взыскать с ООО «Вектор» неустойку за просрочку оплаты в размере 
   50 000 (пятьдесят тысяч) рублей.

3. Взыскать с ООО «Вектор» расходы по уплате госпошлины в размере 
   7 000 (семь тысяч) рублей.
```

### Приложения
```
Приложения:
1. Копия договора № ___ от «___» ______ 20__ г.
2. Копия акта сдачи-приёмки от «___» ______ 20__ г.
3. Копия претензии от «___» ______ 20__ г.
4. Копия квитанции об уплате госпошлины.
5. Выписка из ЕГРЮЛ на Истца.
6. Выписка из ЕГРЮЛ на Ответчика.

«___» ______ 20__ г.

[Подпись] _________________ / [Расшифровка] _________________
```

## Чек-лист проверки оформления

### Для договоров
- [ ] Шрифт 14 pt, интервал 1,5
- [ ] Нумерация страниц
- [ ] Все страницы прошиты (для многостраничных)
- [ ] Подписи с расшифровками
- [ ] Печати (если требуется)
- [ ] Даты проставлены

### Для исков
- [ ] Шапка с реквизитами суда и сторон
- [ ] Цена иска указана
- [ ] Госпошлина уплачена
- [ ] Все приложения в копиях
- [ ] Подпись с расшифровкой
- [ ] Дата составления

### Для претензий
- [ ] Направлена заказным письмом с описью
- [ ] Срок ответа указан
- [ ] Копии документов приложены
- [ ] Контакты для связи указаны

## Электронное оформление

### PDF-документы
- Формат: PDF/A (для долгосрочного хранения)
- Разрешение сканирования: 300 DPI минимум
- Размер файла: до 10 МБ (для подачи в суд онлайн)

### Электронная подпись
- Квалифицированная электронная подпись (КЭП)
- Сертификат действителен на дату подписания
- Проверка подписи перед подачей

---

> ⚠️ **Важно:** Требования могут отличаться в разных судах. Перед подачей проверяй актуальные требования на сайте конкретного суда.
