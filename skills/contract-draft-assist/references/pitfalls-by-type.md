# Частые ошибки по типам документов

## Договоры

### Ошибка 1: Неопределённый предмет
**Проблема:**
```
❌ «Исполнитель оказывает услуги, а Заказчик их оплачивает»
```

**Почему ошибка:** Предмет не конкретизирован, договор может быть признан незаключённым (ст. 432 ГК РФ).

**Исправление:**
```
✅ «Исполнитель оказывает услуги по уборке офисного помещения 
Заказчика площадью 100 кв.м. по адресу: г. Москва, ул. Ленина, д. 1»
```

---

### Ошибка 2: Отсутствие срока действия
**Проблема:**
```
❌ Договор заключён, но срок не указан
```

**Почему ошибка:** Для некоторых договоров срок является существенным условием.

**Исправление:**
```
✅ «Договор вступает в силу с момента подписания и действует до 
«31» декабря 2026 г. или до полного исполнения сторонами своих обязательств»
```

---

### Ошибка 3: Неправильная неустойка
**Проблема:**
```
❌ «Неустойка 10% в день от суммы договора»
```

**Почему ошибка:** Суд снизит по ст. 333 ГК РФ как явно несоразмерную.

**Исправление:**
```
✅ «Неустойка 0,1% в день от суммы задолженности за каждый день просрочки»
```

---

### Ошибка 4: Отсутствие порядка приёмки
**Проблема:**
```
❌ Нет пункта об акте сдачи-приёмки
```

**Почему ошибка:** Невозможно доказать факт выполнения работ.

**Исправление:**
```
✅ «Приёмка работ оформляется Актом по форме Приложения №1. 
Заказчик обязан подписать Акт или направить мотивированный отказ 
в течение 5 дней»
```

---

### Ошибка 5: Неправильная подсудность
**Проблема:**
```
❌ «Все споры рассматриваются в суде по месту нахождения Истца» 
(для договора между двумя юрлицами)
```

**Почему ошибка:** Для юрлиц и ИП споры подведомственны арбитражным судам.

**Исправление:**
```
✅ «Все споры рассматриваются в Арбитражном суде города Москвы»
```

---

## Исковые заявления

### Ошибка 1: Неправильная подсудность
**Проблема:** Иск подан в суд общей юрисдикции, когда должен быть в арбитраж.

**Почему ошибка:** Спор между юрлицами/ИП — компетенция арбитражного суда (ст. 27 АПК РФ).

**Исправление:** Проверить статус сторон:
- Юрлица и ИП → Арбитражный суд
- Физлица без статуса ИП → Суд общей юрисдикции

---

### Ошибка 2: Не соблюдён досудебный порядок
**Проблема:**
```
❌ Иск подан без отправки претензии
```

**Почему ошибка:** По экономическим спорам досудебный порядок обязателен (ст. 4 АПК РФ).

**Исправление:**
1. Отправить претензию заказным письмом с описью
2. Дождаться 30 дней или ответа
3. Приложить копию претензии и квитанцию к иску

---

### Ошибка 3: Неверный расчёт госпошлины
**Проблема:**
```
❌ Госпошлина рассчитана неправильно
```

**Почему ошибка:** Иск оставят без движения (ст. 128 АПК РФ).

**Исправление:** Использовать калькулятор на сайте суда:
- До 100 000 руб.: 4%, мин. 2 000 руб.
- 100 001 - 200 000 руб.: 4 000 + 3% от суммы свыше 100 000
- 200 001 - 1 000 000 руб.: 7 000 + 2% от суммы свыше 200 000
- Свыше 1 000 000 руб.: 25 000 + 1% от суммы свыше 1 000 000, макс. 200 000

---

### Ошибка 4: Отсутствие доказательств
**Проблема:**
```
❌ Иск без приложений (договор, акты, претензия)
```

**Почему ошибка:** Иск оставят без движения (ст. 128 АПК РФ).

**Исправление:** Приложить копии:
- Договора
- Актов выполненных работ
- Претензии с квитанцией об отправке
- Расчёта суммы иска
- Госпошлины
- Выписки из ЕГРЮЛ на стороны

---

### Ошибка 5: Непонятные требования
**Проблема:**
```
❌ «Прошу защитить мои нарушенные права»
```

**Почему ошибка:** Требования должны быть конкретными (ст. 125 АПК РФ).

**Исправление:**
```
✅ «Прошу взыскать с Ответчика в пользу Истца:
- Задолженность: 500 000 рублей
- Неустойку: 50 000 рублей
- Расходы по госпошлине: 7 000 рублей»
```

## Претензии

### Ошибка 1: Неправильный адрес
**Проблема:**
```
❌ Претензия отправлена по фактическому адресу
```

**Почему ошибка:** Юридически значимые сообщения отправляются по адресу из ЕГРЮЛ.

**Исправление:** Проверить адрес в ЕГРЮЛ на сайте nalog.ru

---

### Ошибка 2: Отсутствие срока ответа
**Проблема:**
```
❌ «Просим оплатить в ближайшее время»
```

**Почему ошибка:** Невозможно доказать соблюдение досудебного порядка.

**Исправление:**
```
✅ «Требуем оплатить задолженность в течение 30 (тридцати) 
календарных дней с момента получения настоящей претензии»
```

---

### Ошибка 3: Неполный комплект документов
**Проблема:**
```
❌ Претензия без копий документов
```

**Почему ошибка:** Должник может не понять основание требований.

**Исправление:** Приложить копии:
- Договора
- Актов
- Счёта на оплату
- Расчёта неустойки

---

## Ходатайства

### Ошибка 1: Отсутствие обоснования
**Проблема:**
```
❌ «Прошу назначить экспертизу» без объяснения почему
```

**Почему ошибка:** Суд откажет (ст. 82 АПК РФ требует обоснования).

**Исправление:**
```
✅ «Прошу назначить строительно-техническую экспертизу, поскольку 
для оценки качества выполненных работ требуются специальные познания 
в области строительства»
```

---

### Ошибка 2: Неправильная формулировка вопросов
**Проблема:**
```
❌ «Прошу оценить качество работ»
```

**Почему ошибка:** Вопросы должны быть конкретными.

**Исправление:**
```
✅ «Прошу поставить перед экспертом вопросы:
1. Соответствуют ли выполненные работы требованиям СНиП?
2. Какова стоимость устранения выявленных недостатков?»
```

---

## Обеспечительные меры

### Ошибка 1: Отсутствие встречного обеспечения
**Проблема:**
```
❌ Ходатайство об обеспечительных мерах без встречного обеспечения
```

**Почему ошибка:** Суд может отказать (ст. 94 АПК РФ).

**Исправление:**
```
✅ «Готов предоставить встречное обеспечение в размере 500 000 рублей 
путём внесения денежных средств на депозитный счёт суда»
```

---

## Сроки

### Ошибка 1: Пропуск срока исковой давности
**Проблема:**
```
❌ Иск подан через 5 лет после нарушения
```

**Почему ошибка:** Общий срок исковой давности — 3 года (ст. 196 ГК РФ).

**Исправление:** Проверить срок до подачи иска. Если пропущен — ходатайствовать 
о восстановлении срока при уважительных причинах.

---

### Ошибка 2: Неправильный срок исполнения
**Проблема:**
```
❌ «В течение 10 дней» без указания от какой даты
```

**Почему ошибка:** Невозможно определить момент нарушения.

**Исправление:**
```
✅ «В течение 10 (десяти) рабочих дней с момента получения настоящего уведомления»
```

---

## Чек-лист предотвращения ошибок

### Перед подписанием договора
- [ ] Предмет конкретизирован
- [ ] Срок действия указан
- [ ] Цена и порядок оплаты ясны
- [ ] Ответственность предусмотрена
- [ ] Порядок приёмки описан
- [ ] Подсудность указана верно

### Перед подачей иска
- [ ] Подведомственность проверена (арбитраж или СОЮ)
- [ ] Досудебный порядок соблюдён
- [ ] Госпошлина рассчитана верно
- [ ] Все документы приложены
- [ ] Требования конкретны
- [ ] Срок исковой давности не пропущен

### Перед отправкой претензии
- [ ] Адрес из ЕГРЮЛ проверен
- [ ] Срок ответа указан
- [ ] Копии документов приложены
- [ ] Отправка заказным письмом с описью

---

> ⚠️ **Важно:** Данный список не исчерпывающий. Перед использованием документа обязательна проверка квалифицированным юристом.
