# Обязательные реквизиты документов

## Договоры (ГК РФ)

### Существенные условия (ст. 432 ГК РФ)
Без этих условий договор считается незаключённым:

1. **Предмет договора** — что именно является объектом соглашения
2. **Стороны** — кто заключает договор
3. **Цена** (для возмездных договоров) — стоимость работ/услуг/товара
4. **Срок** (если указан в законе как существенное условие)

### Обязательные реквизиты письменного договора

```
1. Наименование документа («Договор», «Контракт»)
2. Номер и дата заключения
3. Место заключения (город)
4. Преамбула:
   - Полные наименования сторон
   - ФИО представителей
   - Основания полномочий (Устав, доверенность)
5. Предмет договора
6. Права и обязанности сторон
7. Цена и порядок расчётов
8. Сроки исполнения
9. Ответственность сторон
10. Порядок расторжения
11. Срок действия договора
12. Подписи сторон с реквизитами:
    - Наименование организации/ФИО ИП
    - ИНН/КПП
    - Адрес
    - Банковские реквизиты
    - Подпись / Расшифровка
```

## Исковые заявления (АПК РФ ст. 125, ГПК РФ ст. 131)

### Обязательные элементы иска в арбитраж (ст. 125 АПК РФ)

```
1. Наименование арбитражного суда
2. Наименование истца:
   - Для юрлиц: полное наименование, адрес, ИНН/ОГРН
   - Для ИП: ФИО, адрес регистрации, ИНН/ОГРНИП
3. Наименование ответчика:
   - Те же данные, что для истца
4. Цена иска (если подлежит оценке)
5. Обстоятельства дела:
   - Фактические обстоятельства
   - Нормативное обоснование
   - Доказательства
6. Требования к ответчику (в рублях или действия)
7. Расчёт взыскиваемой суммы
8. Сведения о досудебном порядке
9. Список прилагаемых документов
10. Подпись истца/представителя
```

### Обязательные элементы иска в суд общей юрисдикции (ст. 131 ГПК РФ)

```
1. Наименование суда
2. Данные истца:
   - ФИО, адрес, телефон
   - Для представителей: данные доверенности
3. Данные ответчика:
   - ФИО, адрес (если известен)
4. Суть нарушения:
   - В чём заключается нарушение прав
   - Обстоятельства дела
5. Требования и их обоснование
6. Цена иска (если подлежит оценке)
7. Сведения о досудебном порядке
8. Список прилагаемых документов
9. Подпись и дата
```

## Претензии (досудебные)

### Обязательные реквизиты

```
1. Кому: [Наименование должника, адрес]
2. От кого: [Наименование кредитора, адрес, контакты]
3. Заголовок: «ПРЕТЕНЗИЯ»
4. Основание: [Договор №___ от ___]
5. Обстоятельства:
   - Что было нарушено
   - Когда и как
6. Требования:
   - Конкретная сумма или действие
   - Срок исполнения
7. Предупреждение о суде:
   - «В случае неисполнения будем вынуждены обратиться в суд»
8. Приложения: копии документов
9. Подпись, дата, контакты
```

## Ходатайства (АПК РФ гл. 13, ГПК РФ гл. 13)

### Обязательные элементы

```
1. Наименование суда
2. Номер дела
3. Наименование стороны (заявителя)
4. Наименование ходатайства:
   - «Ходатайство об истребовании доказательств»
   - «Ходатайство о назначении экспертизы»
   - и т.д.
5. Обоснование:
   - Почему это необходимо
   - Ссылки на статьи АПК/ГПК
6. Просительная часть:
   - «Прошу...»
7. Подпись и дата
```

## Проверка реквизитов (чек-лист)

### Для договоров
- [ ] Предмет определён достаточно конкретно
- [ ] Стороны идентифицированы
- [ ] Цена указана цифрами и прописью
- [ ] Сроки конкретны (даты или период)
- [ ] Ответственность предусмотрена
- [ ] Подписи с расшифровками
- [ ] Печати (если требуется по уставу)

### Для исков
- [ ] Суд указан верно (подведомственность/подсудность)
- [ ] Все стороны идентифицированы
- [ ] Цена иска рассчитана правильно
- [ ] Нормы права процитированы
- [ ] Доказательства приложены
- [ ] Копии для ответчика есть
- [ ] Квитанция госпошлины приложена

### Для претензий
- [ ] Направлена по адресу из ЕГРЮЛ
- [ ] Способ отправки: заказное письмо с описью
- [ ] Срок ответа указан (обычно 30 дней)
- [ ] Копии документов приложены

## Частые ошибки

| Ошибка | Последствие | Как избежать |
|--------|-------------|--------------|
| Нет предмета договора | Договор незаключён | Чётко описать объект |
| Не указана цена | Споры об оплате | Указать цифрами и прописью |
| Неправильная подсудность | Возврат иска | Проверить ст. 35-38 АПК |
| Нет доказательств досудебного порядка | Оставление без движения | Приложить копию претензии |
| Неверный расчёт госпошлины | Возврат иска | Использовать калькулятор на сайте суда |
