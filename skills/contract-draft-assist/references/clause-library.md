# Библиотека типовых пунктов (Clause Library)

## Неустойка (ст. 330, 332 ГК РФ)

### Вариант 1: Процентная неустойка
```
За каждый день просрочки исполнения обязательства Исполнитель уплачивает 
Заказчику неустойку в размере 0,1% (ноль целых одна десятая процента) 
от суммы неисполненного обязательства за каждый день просрочки.
```

### Вариант 2: Фиксированная неустойка
```
В случае нарушения срока выполнения работ Исполнитель уплачивает Заказчику 
штраф в размере 10 000 (десять тысяч) рублей за каждый факт нарушения.
```

### Вариант 3: Прогрессивная неустойка
```
При просрочке исполнения обязательства:
- до 10 дней — 0,1% в день
- от 11 до 30 дней — 0,3% в день
- свыше 30 дней — 0,5% в день
```

**Примечание:** Согласно ст. 333 ГК РФ, суд вправе снизить неустойку, если она явно несоразмерна последствиям нарушения.

## Форс-мажор (ст. 401 ГК РФ)

### Стандартный пункт
```
Стороны освобождаются от ответственности за частичное или полное 
неисполнение обязательств по Договору, если оно вызвано обстоятельствами 
непреодолимой силы (форс-мажор): пожар, наводнение, землетрясение, 
военные действия, акты государственных органов.

Сторона, неспособная исполнить обязательства, обязана уведомить другую 
Сторону в течение 5 (пяти) дней с момента наступления таких обстоятельств.

Если форс-мажор продолжается более 30 (тридцати) дней, любая Сторона 
вправе расторгнуть Договор в одностороннем порядке.
```

## Подсудность и применимое право

### Для арбитражных судов
```
Все споры, возникающие из настоящего Договора или в связи с ним, 
подлежат рассмотрению в Арбитражном суде города Москвы.

К правоотношениям Сторон применяется право Российской Федерации.
```

### Для судов общей юрисдикции
```
Все споры, возникающие из настоящего Договора, подлежат рассмотрению 
в суде по месту нахождения Истца (выбрать нужное).

К правоотношениям Сторон применяется право Российской Федерации.
```

### Альтернативная подсудность (по выбору истца)
```
Споры по настоящему Договору рассматриваются судом по выбору Истца:
- по месту нахождения Ответчика, или
- по месту исполнения Договора, или
- по месту заключения Договора.
```

## Конфиденциальность

### Стандартный пункт
```
Стороны обязуются сохранять конфиденциальность всей информации, 
полученной в ходе исполнения Договора, и не разглашать её третьим 
лицам без письменного согласия другой Стороны.

Обязательство конфиденциальности действует в течение 3 (трёх) лет 
с момента окончания срока действия Договора.

Исключения:
- Информация, ставшая общедоступной
- Информация, полученная от третьих лиц без обязательств
- Раскрытие по требованию государственных органов
```

### NDA с штрафом
```
За разглашение конфиденциальной информации нарушившая Сторона уплачивает 
штраф в размере 500 000 (пятьсот тысяч) рублей за каждый факт нарушения, 
а также возмещает все причинённые убытки в полном объёме.
```

## Порядок приёмки работ

### Акт сдачи-приёмки
```
По завершении работ Исполнитель направляет Заказчику Акт сдачи-приёмки 
выполненных работ по форме Приложения №1.

Заказчик обязан в течение 5 (пяти) рабочих дней:
- Подписать Акт без замечаний, или
- Направить мотивированный отказ с перечнем недостатков

При отсутствии подписанного Акта или отказа в указанный срок, 
работы считаются принятыми без замечаний.
```

### Устранение недостатков
```
Если Заказчик выявил недостатки в работах, Исполнитель обязан 
устранить их своими силами и за свой счёт в течение 10 (десяти) 
рабочих дней с момента получения уведомления.
```

## Расторжение договора

### По соглашению сторон
```
Настоящий Договор может быть расторгнут по соглашению Сторон, 
оформленному дополнительным соглашением.
```

### Односторонний отказ (ст. 310 ГК РФ)
```
Заказчик вправе расторгнуть Договор в одностороннем порядке, 
уведомив Исполнителя за 30 (тридцать) календарных дней.

При таком расторжении Заказчик оплачивает фактически выполненные 
работы на дату расторжения.
```

### Расторжение через суд (ст. 450 ГК РФ)
```
При существенном нарушении Договора одной из Сторон, другая Сторона 
вправе требовать расторжения Договора в судебном порядке.

Существенным признается нарушение, которое влечёт для другой 
Стороны такой ущерб, что она в значительной степени лишается 
того, на что была вправе рассчитывать.
```

## Ответственность и ограничение

### Ограничение ответственности
```
Общая ответственность Сторон по настоящему Договору ограничена 
суммой 1 000 000 (один миллион) рублей.

Стороны не несут ответственности за упущенную выгоду, косвенные 
или последующие убытки.
```

### Возмещение убытков (ст. 15 ГК РФ)
```
Сторона, нарушившая обязательства, обязана возместить другой 
Стороне причинённые убытки в полном объёме, включая реальный 
ущерб и упущенную выгоду.
```

## Разрешение споров

### Досудебный порядок (обязательный по ст. 4 АПК РФ)
```
Все споры по настоящему Договору подлежат досудебному урегулированию 
путём направления письменной претензии.

Срок ответа на претензию: 30 (тридцать) календарных дней.

Обращение в суд возможно только после соблюдения досудебного порядка.
```

### Медиация
```
При невозможности урегулировать спор в досудебном порядке, Стороны 
обязуются рассмотреть возможность применения процедуры медиации 
до обращения в суд.
```

---

## Заключительные положения

### Изменения и дополнения
```
Все изменения и дополнения к настоящему Договору действительны 
только в письменной форме и должны быть подписаны уполномоченными 
представителями обеих Сторон.
```

### Передача прав (цессия)
```
Уступка прав требований по настоящему Договору возможна только с 
письменного согласия другой Стороны.
```

### Notices (уведомления)
```
Все уведомления по Договору направляются:
- Заказным письмом с уведомлением по почтовому адресу, или
- Курьерской доставкой под подпись, или
- Электронной почтой с подтверждением прочтения.

Уведомление считается полученным на 3-й день после отправки.
```

## Специфичные пункты по договорам

### Для договора подряда (ст. 702-768 ГК РФ)
```
Материалы и оборудование:
- Если материалы предоставляет Заказчик — указать перечень и сроки
- Если материалы предоставляет Подрядчик — включить в цену

Гарантийный срок: __ месяцев/лет с момента сдачи работ.
```

### Для договора аренды (ст. 606-670 ГК РФ)
```
Обеспечительный платёж: Арендатор вносит сумму в размере ___ 
в качестве обеспечения исполнения обязательств.

Право преимущественной аренды: Арендатор имеет преимущественное 
право на заключение договора на новый срок.
```

### Для договора поставки (ст. 454-534 ГК РФ)
```
Порядок поставки:
- Партиями по заявкам Покупателя
- Самовывоз со склада Поставщика
- Доставка транспортом Поставщика

Переход права собственности: с момента передачи товара.
```
