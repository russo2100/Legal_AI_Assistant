# Структура юридического ответа

## Обязательные разделы

### 1. КРАТКИЙ ВЫВОД (2-3 предложения)

**Назначение:** Дать прямой ответ на вопрос пользователя.

**Требования:**
- Конкретика без воды
- Ссылка на ключевую норму
- Оценка перспектив (если применимо)

**Пример:**
```
КРАТКИЙ ВЫВОД:
Заказчик вправе требовать уплаты неустойки по ст. 330 ГК РФ за просрочку 
выполнения работ. Размер неустойки определяется договором или законом. 
Судебная практика по аналогичным спорам преимущественно на стороне заказчика.
```

---

### 2. ПРИМЕНИМЫЕ НОРМЫ

**Назначение:** Обосновать ответ ссылками на законодательство.

**Формат:**
```
1. [Название кодекса] [Статья]
   [Цитата или краткое содержание]
   [Источник](URL)
```

**Требования:**
- Только релевантные статьи (3-5 максимум)
- Актуальная редакция
- Кликабельные ссылки на Consultant.ru или Гарант

**Пример:**
```
ПРИМЕНИМЫЕ НОРМЫ:

1. ГК РФ Статья 330
   «Неустойкой (штрафом, пеней) признается определенная законом или 
   договором денежная сумма, которую должник обязан уплатить кредитору 
   в случае неисполнения или ненадлежащего исполнения обязательства...»
   [Источник](https://www.consultant.ru/document/cons_doc_LAW_5142/)

2. ГК РФ Статья 332
   «Законная неустойка может быть установлена только законом...»
   [Источник](https://www.consultant.ru/document/cons_doc_LAW_5142/)
```

---

### 3. СУДЕБНАЯ ПРАКТИКА

**Назначение:** Показать, как суды применяют нормы в аналогичных делах.

**Формат:**
```
1. Дело № [номер]
   [Краткое описание: стороны, суть спора, результат]
   [Источник](URL на kad.arbitr.ru)
```

**Требования:**
- 2-5 релевантных дел
- Дела за последние 2-3 года
- Ссылки на полные тексты решений

**Пример:**
```
СУДЕБНАЯ ПРАКТИКА:

1. Дело № А40-123456/2024
   ООО «Ромашка» (заказчик) v. ООО «Вектор» (подрядчик)
   Спор о взыскании неустойки за просрочку выполнения работ по договору подряда.
   Суд удовлетворил требования частично: взыскана неустойка в размере 50% от заявленной.
   [Решение](https://kad.arbitr.ru/Card/123456)

2. Дело № А41-789012/2023
   ИП Петров v. ООО «СтройМастер»
   Взыскание неустойки за нарушение сроков ремонта.
   Суд удовлетворил требования полностью.
   [Решение](https://kad.arbitr.ru/Card/789012)
```

### 4. РИСКИ И АЛЬТЕРНАТИВЫ

**Назначение:** Предупредить о возможных проблемах и предложить варианты.

**Формат:**
```
РИСКИ:
- [Риск 1]
- [Риск 2]

АЛЬТЕРНАТИВЫ:
- [Вариант 1]
- [Вариант 2]
```

**Пример:**
```
РИСКИ:
- Суд может снизить неустойку по ст. 333 ГК РФ, если она явно несоразмерна
- Пропуск срока исковой давности (3 года по ст. 196 ГК РФ)
- Отсутствие доказательств факта нарушения обязательства

АЛЬТЕРНАТИВЫ:
- Претензионный порядок (экономия времени и средств)
- Медиация (внесудебное урегулирование)
- Взыскание убытков вместо неустойки (ст. 394 ГК РФ)
```

---

### 5. ДИСКЛЕЙМЕР (ОБЯЗАТЕЛЬНО)

**Текст:**
```
⚠️ **Дисклеймер:** Аналитическая подсказка. Требуется проверка юристом.

Данная информация носит справочный характер и не является официальной 
юридической консультацией. Для принятия конкретных мер обратитесь к 
квалифицированному юристу с документами по вашему делу.
```

## Стилистические требования

### Язык и тон
- **Профессиональный**, но понятный
- Избегай канцеляризмов («надлежащий», «вышеуказанный»)
- Используй активный залог («суд взыскал», а не «было взыскано»)

### Форматирование
- **Жирный** для заголовков и ключевых терминов
- `Код` для номеров статей и дел
- [Ссылки](URL) кликабельны

### Объём
- Краткий вывод: 2-3 предложения
- Нормы: 3-5 статей
- Практика: 2-5 дел
- Общий объём: 500-1500 слов

---

## Чек-лист перед отправкой

- [ ] Все разделы присутствуют
- [ ] Ссылки кликабельны и ведут на авторитетные источники
- [ ] Дисклеймер в конце
- [ ] Нет обещаний гарантированного результата
- [ ] Упомянуты риски и альтернативы
- [ ] Цитаты из норм точные (проверить по источнику)
