# Директория для текстов кодексов РФ

## Как добавить кодексы

1. Скачайте тексты кодексов в формате `.txt` из надёжного источника:
   - Consultant.ru: https://www.consultant.ru/
   - Garant.ru: https://www.garant.ru/
   - Официальный портал: https://pravo.gov.ru/

2. Сохраните файлы в эту папку:
   - `GK.txt` — Гражданский кодекс
   - `APK.txt` — Арбитражный процессуальный кодекс
   - `GPK.txt` — Гражданский процессуальный кодекс
   - `KoAP.txt` — Кодекс об административных правонарушениях

3. Запустите индексацию:
   ```bash
   python src/rag/indexer.py --codes data/codes
   ```

```
ГРАЖДАНСКИЙ КОДЕКС РОССИЙСКОЙ ФЕДЕРАЦИИ

Статья 1. Основные начала гражданского законодательства
1. Гражданское законодательство основывается на признании...

Статья 2. Отношения, регулируемые гражданским законодательством
1. Гражданское законодательство определяет правовое положение...
```
